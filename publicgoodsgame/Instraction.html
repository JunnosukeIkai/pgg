  • 4人1グループのゲームを行います。1ラウンド開始時にそれぞれの参加者に初期保有ポイントとして{{ C.ENDOWMENT }}ポイント与えられます.
  • グループ内で出資可能な共同プロジェクトがあり、参加者は、手持ちの保有ポイントである{{ C.ENDOWMENT }}ポイントから、いくらかを共同プロジェクトへ貢献するのかを決めます.
  • 貢献ポイントは、参加者がそれぞれ任意に決めるができます。また、全く貢献をしないこと（貢献ポイント = 0 ポイント）もできます.
  • 貢献をしなかったポイントは、そのままあなたの手元に残ります。参加者全員の貢献ポイントの総額によって各参加者の公共プロジェクトから得られるポイントが決まります. 貢献ポイントの合計が X ポイントのとき、X × {{ C.MULTIPLIER }}/{{ C.PLAYERS_PER_GROUP }} ポイントです.
  • なお、本実験は10ラウンドの繰り返しが行われます。なお、2ラウンド目以降は一つ前のラウンド末の保有ポイントがラウンド開始時の保有ポイントになります.
Nラウンド末におけるあなたの保有ポイントは次の式で計算されます。

(Nラウンド開始時の保有ポイント) -（あなたの貢献ポイント）＋ (全員の貢献ポイントの合計)×{{ C.MULTIPLIER }} /{{ C.PLAYERS_PER_GROUP }}
